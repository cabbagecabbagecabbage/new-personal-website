Projects

UNO

An online multiplayer UNO clone with an easy-to-use interface. Allows multiple simultaneous game rooms and real-time gameplay with other players. Made with HTML, CSS, JS, Express.js, Socket.io, and SweetAlert2.

LOAF

LOAF (Lightweight & Open-source Anonymous Forum) is a general-purpose anonymous Q&A platform application built with Angular, ASP.NET Core, and SQLite. Supports search, post tags, and LaTex rendering.

Paint Program

A clone of MS Paint with a vast array of features and shortcuts, such as integrated image searching and scraping. Made with Pygame (Python 3) and User Agent.

Discord bot for DMOJ. Displays statistics and leaderboards, fetches problems, officiates duels (2 users racing to solve a problem), and more. Made with discord.py, DMOJ api, and flask. Note: this bot is not officially affiliated with DMOJ.

Arduino Safe

A safe built with Arduino, NodeMCU and other components. Supports password protection, password change, push button to lock door, and Discord notifications when the door is locked/unlocked.
